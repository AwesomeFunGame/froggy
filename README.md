## Develop

### Install
  * [node.js](http://nodejs.org/download/)
  * [Elm 0.13](http://elm-lang.org/Install.elm)

### Check out
    git clone https://github.com/thSoft/froggy.git
    cd froggy
    build/init.sh

### Build
    build/build.js
  * Open `target/index.html`

### Release
    build/publish.js
